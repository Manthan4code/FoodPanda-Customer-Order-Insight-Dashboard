# FoodPanda-Customer-Order-Insight-Dashboard
## Project Overview
This Excel-based dashboard provides a comprehensive analysis of FoodPanda's sales performance, customer acquisition trends, and regional engagement across key cities in Pakistan. The goal of this project was to transform raw transactional data into actionable insights regarding revenue, customer loyalty, and restaurant performance.
## Key Features & Insights

1. High-Level KPIs
Quick glance at the core business metrics:
+ Total Customers: 6,000
+ Total Orders: 9,090
+ Total Revenue: ₹ 72,71,237.07
+ Average Rating: 2.01 (Indicates a potential area for service improvement)

2. Regional Analysis
+ Order Frequency by City: A comparison of order volumes across Islamabad, Karachi, Lahore, Multan, and Peshawar. Peshawar currently leads in order frequency.
+ Active vs. Inactive Customers: A stacked bar chart visualizing customer retention and churn risks across different regions.

3. Growth & Performance
+ New Customers Acquired: A line chart showing acquisition trends from 2023 to 2025. There is a visible peak in 2024, followed by a decline in 2025 that requires strategic attention.
+ Top Restaurants by Customer: Analysis of brand popularity, showing Subway as the top-performing restaurant by customer count, followed by McDonald's and KFC.

## Tools Used
+ Microsoft Excel: Core platform for data processing and visualization.
+ Power Query: For data cleaning and transformation (ETL).
+ Pivot Tables: For data aggregation and summarizing metrics.
+ Excel Charts: Used for the visual components (Line, Bar, and Stacked Bar charts).

## Recommendations Based on Data
+ Improve Ratings: The average rating of 2.01 suggests a need for a quality-of-service audit or better delivery partner training.
+ Re-engage Inactive Users: Specific marketing campaigns should target "Inactive" segments identified in the city-wise bar chart.
+ Investigate 2025 Decline: The drop in customer acquisition for 2025 suggests a need for new promotional strategies or a competitive analysis.
